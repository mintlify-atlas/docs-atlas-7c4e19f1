Claudette is a Sublime Text package that connects your editor to the Anthropic Claude AI API. Instead of switching between your editor and a browser, you can ask questions, get code explanations, and hold multi-turn conversations with Claude in a dedicated chat view — right inside Sublime Text.

## Key features

  <Card title="Ask questions" icon="message" href="/guides/asking-questions">

  <Card title="Manage context" icon="files" href="/guides/managing-context">

## Prerequisites

Before you install Claudette, make sure you have the following:

- **Sublime Text 3 or 4** — Claudette is a Sublime Text package and requires a working Sublime Text installation.
- **An Anthropic API key** — Claudette communicates directly with the Claude API. You need a key from [console.anthropic.com](https://console.anthropic.com/settings/keys) to use it.

<Note>
  API usage is billed by Anthropic based on the number of tokens sent and received. Review [Anthropic's pricing](https://www.anthropic.com/pricing) before you start.
</Note>

## Privacy

Everything you send through Claudette — your questions, selected code, and any files you include as context — is transmitted to Anthropic's servers to generate a response. Review [Anthropic's Privacy & Legal documentation](https://support.anthropic.com/en/collections/4078534-privacy-legal) to understand how your data is handled before sharing sensitive or proprietary code.
